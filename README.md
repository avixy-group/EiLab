# EiLab

A web-based playground to interact with Ei

## Requirements

- Git
- Node.js

## Quickstart

1. Clone this repository

        $ git clone https://github.com/lordriyan/EiLab.git

2. Install all dependecies

        $ yarn

3. Run your project

    - Development

            $ yarn dev

    - Production

            $ yarn build
            $ yarn start

4. Navigate to http://localhost:3000/ to see your web page
